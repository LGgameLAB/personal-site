[TerraMaster](https://terramaster.netlify.app/) is a GeoGuessr clone that I made back in 2021 after 
GeoGuessr decided to become "pay to play". It uses the google maps API was probably my
first work using API. Hardest part was probably calculating the distance between two points on a 
spherical surface.
